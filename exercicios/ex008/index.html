Nesta frase temos um termo em negrito usando a tag <b>

Note: A tag <b> apenas possui forma, não possui semântica

Esta frase possui um termo em destaque usando a tag <strong >

Itálico/ Ênfase

Esta frase possui um termo em italico utilizando a tag <i> (não semântica)

Esta frase possui um termpo em ênfase utilizando a tag <em> (semântica)

Texto Marcado

Podemos criar um texto marcado utilizando a tag <mark>

Altrando a cor padrão da tag <mark>: texto marcado

Texto grande e pequeno

Estamos criano um texto grande e um texto pequeno com as tags <big> e <small>, onde a tag big não é mais tão utilizada

Texto Excluído

Um texto marcado como excluído para indicar que ele deve ser lido mas não considerado, utilizando a tag <del>

Texto Inserido

Marcar o texto como inserido para dar ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>

Texto sobrescrito

Exemplo: x20+3, utilizando a tag <sup>

Texto subescrito

Exemplo: H2O, utilizando a tag <sub>

Recomendação

Forma -> utilizar CSS3
Sentido -> utilizar tags HTML5
